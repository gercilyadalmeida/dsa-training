# Exercise 1: Find Maximum Element

## Description
Write a program that finds the maximum (largest) element in an array.

## Input
First line: an integer N (number of elements)
Second line: N integers separated by spaces

## Output
Print the maximum element in the array.

## Examples
```
Input:
5
3 7 2 9 1
Output: 9

Input:
4
10 5 8 3
Output: 10

Input:
1
42
Output: 42
```

## Constraints
- N >= 1
- All elements are valid integers
