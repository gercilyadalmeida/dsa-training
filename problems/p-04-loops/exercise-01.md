# Exercise 4: Sum of Numbers

## Description
Write a program that calculates the sum of all integers from 1 to N using a loop.

## Input
A single integer N.

## Output
Print the sum of all integers from 1 to N (inclusive).

## Examples
```
Input: 5
Output: 15
(Explanation: 1 + 2 + 3 + 4 + 5 = 15)

Input: 10
Output: 55
(Explanation: 1 + 2 + 3 + ... + 10 = 55)

Input: 1
Output: 1
```

## Constraints
- N is a positive integer
- N >= 1
