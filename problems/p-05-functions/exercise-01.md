# Exercise 1: Add Two Numbers with Function

## Description
Write a program that defines a function to add two numbers and returns their sum.

## Input
Two integers separated by a space.

## Output
Print the sum of the two numbers.

## Examples
```
Input: 5 3
Output: 8

Input: -2 7
Output: 5

Input: 0 0
Output: 0

Input: 100 50
Output: 150
```

## Constraints
- Both numbers are valid integers
- Numbers can be positive, negative, or zero
