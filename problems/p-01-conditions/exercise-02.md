# Exercise 2: Leap Year (Bixextile Year)

## Description
Write a program that determines if a given year is a leap year (bissextile year).

## Input
A single integer representing a year.

## Output
- Print "leap" if the year is a leap year
- Print "not leap" if the year is not a leap year

## Rules
A year is a leap year if:
- It is divisible by 4 AND not divisible by 100, OR
- It is divisible by 400

## Examples
```
Input: 2020
Output: leap

Input: 1900
Output: not leap

Input: 2000
Output: leap

Input: 2023
Output: not leap
```

## Constraints
- Year is a valid positive integer
