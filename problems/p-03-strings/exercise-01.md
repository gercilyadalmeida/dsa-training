# Exercise 1: Reverse a String

## Description
Write a program that reverses a given string.

## Input
A single line containing a string.

## Output
Print the reversed string.

## Examples
```
Input: hello
Output: olleh

Input: world
Output: dlrow

Input: a
Output: a

Input: abc123
Output: 321cba
```

## Constraints
- The string can contain letters, digits, and spaces
- The string length is at least 1
